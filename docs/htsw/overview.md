# HTSW Overview

HTSW is a near zero-abstraction framework and collection of formats for
 representing Housing entities as text.

It consists of two main formats: import.json and HTSL.

## import.json

Importables are the JSON-based configuration format for HTSW. See the
 [Importables](./importables.md) page for details.

## HTSL

HTSL is a basic markup language for [Housing Actions](./htsl/actions.md).

An overview of the syntax can be found [here](./htsl/basic-syntax.md).
